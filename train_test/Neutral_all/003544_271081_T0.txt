С принятием закона у граждан, выплачивающих ипотеку и попавших в трудную жизненную ситуацию, появится возможность взять перерыв, передохнуть, решить свои текущие проблемы. Потеря работы, снижение заработной платы, болезнь, как правило, не являлись для банков уважительной  причиной для установления заемщику «льготного периода» по оплате кредита. Невозможность платить, зачастую,  приводила к потере ипотечного жилья. Теперь кредитно-финансовые организации по заявлению гражданина будут обязаны  предоставить «ипотечные каникулы».  По статистике, поиск нового места работы занимает, в среднем, два — три месяца. Заемщику будет предложено шесть месяцев для решения проблем с работой, восстановления здоровья, платёжеспособности. Инициатива подготовлена по поручению Президента с целью дополнительной защиты  прав заёмщиков по ипотечному кредитованию, кроме этого Владимир Путин поручил предоставить возможность воспользоваться ипотечными каникулами и тем заемщикам, кто взял кредит на покупку жилья до принятия этого законопроекта»
